## 📅 Core Engine Features

### 📈 Life Potential Trajectory Graph
- **Overview**: Continuous Vimshottari Dasha curve mapping age trajectories (20, 25, 29, 35, 45, 60+).
- **Functionality**: Interactive nodes highlighting active Mahadasha/Antardasha periods, growth peaks, consolidation phases, and planetary aspects.

### ✈️ Relocation Flight Path Atlas
- **Overview**: Interactive geographical vector world map mapping birthplace to optimal international relocation destinations.
- **Functionality**: Animated flight path arcs evaluating destination suitability across Career Growth, Wealth Potential, Marriage/Domestic Security, and Mental Peace.

### 🎯 Career Alignment Compass
- **Overview**: 4-quadrant professional suitability matrix mapping planetary strength to modern industries.
- **Functionality**: Categorizes careers into *Most Favorable*, *Favorable*, *Moderate*, and *Least Favorable* based on 10th lord, 9th lord, and D9 Navamsha rules.
